# Markdown Chords

Plugin ID: `markdown-chords`
Links: [GitHub repository](https://github.com/dnotes/obsidian-markdown-chords) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=markdown-chords)
