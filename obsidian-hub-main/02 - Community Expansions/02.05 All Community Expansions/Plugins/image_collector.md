# Image Collector

Plugin ID: `image_collector`
Links: [GitHub repository](https://github.com/tdaykin/obsidian_image_collector) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=image_collector)
Developed by: [[tdaykin]]
