# Rollover Daily Todos

Plugin ID: `obsidian-rollover-daily-todos`
Links: [GitHub repository](https://github.com/lumoe/obsidian-rollover-daily-todos) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=obsidian-rollover-daily-todos)
Developed by: [[lumoe]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

This Obsidian.md plugin rolls over incomplete TODOs from the previous daily note to today's daily note. (https://obsidian.md). (Originally created by Matthew Sessions)
