# Ink

Plugin ID: `ink`
Links: [GitHub repository](https://github.com/daledesilva/obsidian_ink) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=ink)
Developed by: [[daledesilva]]
