# Auto Tasks

Plugin ID: `auto-tasks`
Links: [GitHub repository](https://github.com/jamiefdhurst/obsidian-auto-tasks) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=auto-tasks)
